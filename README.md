# DMN on Rails example

This project is an example app I presented in this talk: http://karreiro.com/assets/talks/2020-dmn-for-developers.pdf 

## Prerequisites

* **Ruby**: `ruby 2.6.3`
* **Rails**: `6.0.3.3`
* **DMN service**: Check the talk slides to understand the setup ;-)
